System Overview

Scoring Pipeline

CIBIL-weighted credit scoring with behavioral feature extraction using sliding window analysis on transaction history.

Risk Analysis

Graph-based cascade detection and Union-Find clustering for systemic risk assessment across borrower networks.

Capital Allocation

0/1 Knapsack dynamic programming to maximize expected revenue within capital budget constraints.

Performance

Segment Tree and TreeMap for O(log n) range queries on 1,000 customer dataset.

DSA Algorithms Used

TreeMap Red-Black Tree for sorted score indexing and percentile queries
Deque Sliding window over transaction history for feature extraction
SegmentTree Range sum/max queries on credit limits O(log n)
Graph (BFS/DFS) Cascade detection and 3-color cycle detection
Union-Find Borrower clustering with path compression and union by rank
Knapsack DP 0/1 optimization for capital allocation O(N×W)
